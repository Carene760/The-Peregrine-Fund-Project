# Documentation du module Android App

Ce dossier regroupe la documentation technique de l application Android.

## Structure

```text
documentation/app/
├── README.md
├── GUIDE_TECHNIQUE_UTILISATION.md
├── captures/
└── v1/
    ├── README.md
    ├── EXECUTION_v1.md
    ├── ARCHITECTURE_v1.md
```

## Regle de versioning

- v1 represente le snapshot documentaire initial.
- une evolution majeure du module Android cree un dossier v2.
- les corrections mineures peuvent rester en v1 avec trace dans CHANGELOG.
